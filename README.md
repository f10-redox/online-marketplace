# Online Marketplace

A simple e-commerce website built using **HTML**, **CSS**, and **JavaScript**. This website allows users to browse products, add them to a shopping cart, view the cart, and clear it.

## Features

- **Product Listings**: Display a list of products with prices and "Add to Cart" buttons.
- **Cart Functionality**: Add products to the cart, view the cart, and see the total price.
- **Clear Cart**: Clear all items from the cart.
- **Responsive Design**: The website is fully responsive, so it works well on both desktop and mobile devices.

## Tech Stack

- **HTML**: Structure and content of the website.
- **CSS**: Styling the page to make it look visually appealing.
- **JavaScript**: Logic for adding products to the cart, updating the cart total, and managing local storage.

## How to Use

1. Open `index.html` to view the homepage with product listings.
2. Click on the "Add to Cart" button to add items to the cart.
3. Navigate to the `cart.html` page to view the items in your cart and the total cost.
4. Use the "Clear Cart" button to remove all items from the cart.
